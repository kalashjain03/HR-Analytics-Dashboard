# HR-Analytics-Dashboard

## 1. Business HR Analytics Dashboard
An interactive Power BI dashboard designed to analyze employee attrition, workforce demographics, salary distribution, and job role trends to support strategic HR decision-making.

## 2. Description
The HR Insights Pro Dashboard is a comprehensive Power BI solution built to analyze workforce data and uncover key drivers of employee attrition. It provides insights into employee demographics, salary bands, job roles, departments, and tenure to help organizations understand where and why turnover occurs.
With interactive filters and dynamic KPIs, the dashboard enables HR leaders to monitor attrition rates, compare departments, identify high-risk roles, and make data-driven decisions to improve employee retention, optimize compensation strategies, and strengthen overall workforce planning.

## 4. Data Source
Source: IBM HR Analytics Employee Attrition & Performance Dataset (public sample dataset available on Kaggle).
The dataset contains records for 1,470 employees, including:
Employee demographics (Age, Gender, Education)
Compensation (Monthly Salary Bands)
Job Roles and Departments (HR, R&D, Sales)
Years at Company
Attrition status

The data is structured in a single primary employee table with categorical and numerical attributes enabling segmentation and cross-filtering.
